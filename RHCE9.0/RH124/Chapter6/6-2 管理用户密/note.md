##  用户密码的相关信息
- 在/etc/shadow文件中的每一行都包含一个用户的信息，例如：<br>
 ***user03 : $asdhajksdhjkhjkashkdja : 17933 : 0 : 99999 : 7 : 2 : 18113 ：***
- user03:此用户的用户名
- $asdhajksdhjkhjkashkdja：加密后的密码
- 17933：上次修改密码的时间。从1970-01-01开始计算
- 0：密码的最小有效期
- 99999：密码的最大有效期
- 7：密码到期前的警告日期
- 2：密码到期后的宽限日期
- 18113：账号的失效日期。从1970-01-01开始计算
-  ：保留使用
