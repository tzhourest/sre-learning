# 学习笔记
日期：2026-05-28
主题：正则表达式

## 匹配行的开始和结束位置
- ^只可以匹配行的开头
- $只可以匹配行的结尾

## 正则表达式中通配符以及乘数的使用
- .：符号为通配符，可以代替任意单个字符
- ?：前面这一项是可选的，并且最多只能匹配一次
- []：可以匹配中括号里的任意字符
- *：前面这一项可以被匹配零次或多次
- +：前面的这一项被匹配了一次或多次
- {n}：前面的项目被完全匹配了n次
- {n,}：前面的项目被匹配了1次或更多
- {,m}：前面的这一项最多可以重复出现m次
- {n,m}：起那么的这一项至少被匹配了n次但不超过m次
- [:alnum:]字母数字字符与[0-9A-Za-z]想同
